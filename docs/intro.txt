parcel - библиотека для парсинга текстовой информации в любом её представлении.

Пользователь сам описывает, как он хочет обработать текстовый блок на основе parcel - языка.
Язык parcel включает:
    - зарезервированные теги: базовый набор тегов для описания текстового содержимого
    - хуки: используются для получения данных
    - определения (vardef): строительный блок, использутся для описания текстовых блоков и переиспользования в хуках/функциях/определениях
    - функции: встроенные функции для обработки и фильтрации данных
